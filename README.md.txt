# Mini Project 1 - User Transaction System

Description:
A basic JavaScript system that simulates users, accounts,
transactions, balance processing, and system summaries.

Features:
- User classification (Adult/Child)
- Transaction processing
- Income calculation
- Expense calculation
- Final balance calculation
- System-wide summary

Tech Stack:
- JavaScript
- Node.js

Concepts Used:
- Arrays
- Objects
- Functions
- Loops
- Conditions
- Logic building
- Nested loops
- Data processing
